# speedtester.sh

## Download
```shell
wget https://raw.githubusercontent.com/spy-duck/speedtester.sh/refs/heads/main/speedtester.sh && chmod +x ./speedtester.sh
```

## Options

| Option      | Description                                     |
| ----------- | ----------------------------------------------- |
| `-n NUMBER` | Number of iterations (default: 5)               |
| `-i NUMBER` | Interval between tests in seconds (default: 30) |
| `-s ID`     | Specific Speedtest server ID                    |
| `-h`        | Show help                                       |


## Run
## Download
```shell
./speed-tester.sh -n 8 -i 60
```
